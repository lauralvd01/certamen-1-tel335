# Node template repo

Considere el código de base. Realize el `fork` de este repositorio en su cuenta `github`.

Para levantar, debe primero instalar las dependencias:

```
npm install
```

Y luego ejecutar:

```
npm run start
```

o también `npm run start:dev` para utilizar `nodemon`.
